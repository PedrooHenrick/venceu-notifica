# VencimentoSST

Controle simples e visual de vencimentos de documentos de SST. Empresas, funcionários, ASOs, NRs e licenças sempre em dia.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (banco + autenticação)
- Shadcn/ui

## Scripts

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Gera build de produção
- `npm run preview` - Visualiza build localmente
